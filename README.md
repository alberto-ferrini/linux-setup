# Linux Setup

Setup script for Linux environment.

The script installs Git, Java (JDK) and configures the PATH environment variable.

## Installation

For the installation, launch the file [setup](./setup).

```bash
bash setup
```

## Contents

### .setup file

This file runs at login and configures the environment variables.

### scripts

This directory contains the scripts added to PATH environment variable.

### scripts/mvnw

This script executes mvnw inside a Maven Project.
